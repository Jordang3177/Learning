• Primary Keys are a unique value that represents a whole row.
  • Foreign Keys are keys that exist in one table but refers to a primary key in another table.
  • Attributes are columns headings in th erelational table, while tuples are meant to describe the rows in the table.
  • SQl stands for Structured Query Language and it is used as the language for Relational Database Management Systems.
  • One example of a primary key that you could use would be a person's driver's license number since it is unique to the person, or if you are in a professional setting then you can have it be the employee's ID, since you are only going to have a database of employees these should be unique as well in that company.
